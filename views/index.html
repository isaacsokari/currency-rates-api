Exchange Rates API

(Assessment)

Example

To use the API, use this format

{base_url}/api/rates?base={base currency}&currency={target currencies}

eg. {base_url}/api/rates?base=USD&currency=CAD,GBP,EUR

Explanation

The example endpoint URL takes "USD" as the base currency and it expects to get the exchange rates for

  • USD -> CAD
  • USD -> GBP
  • USD -> EUR
